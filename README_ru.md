### Описание
- Чистая архитектура;
- Корректное завершение работы приложения;
- Логирование ошибок в Stderr и в файл;
- Все виды тестирования в staging с доступом в docker network (Github Actions);
- Сборка образа и развертывание (Github Actions);
- Применение миграций;
- Маленький размер образа (alpine).
