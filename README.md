### Features
- Clean architecture;
- Graceful application shutdown;
- Error logging to Stderr and a file;
- All types of testing in staging with access to the Docker network (Github Actions);
- Image building and deployment (Github Actions);
- Migration application;
- Small final image size (alpine).
